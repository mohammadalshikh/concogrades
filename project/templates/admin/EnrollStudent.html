Enroll students using file drop:
File format/content restrictions:
- File drop accepts .txt files only.
- Entries cannot be assessment grades (student's personal info only).
- Entries must be in order: Student ID, First name, Last name, Password.
- Entries must be seperates by a space or tab.

Example: 

  123456 First Second xf38d12
How to upload:
1- Save the file on your local drive.
2- Upload the file containing students info then click "Upload Text File".
